# bilibili_live_tool (重构版)

> ⚠️ **开发中分支** — 本分支正在进行 Textual 重构，目前仅 `src/utils/` 为完成代码，其余部分为占位模板。

基于 Textual 重构的 B 站直播终端工具（TUI），支持开播/下播、修改标题/分区、查看直播间弹幕等功能。

## 声明

本程序仅用于声明的功能，用户使用本程序产生的账号问题，与本程序无关。

## 安装/运行

### 源码运行

```bash
# 克隆仓库并切换到 dev 分支
git clone https://github.com/chenxi-Eumenides/bilibili_live_tool.git
cd bilibili_live_tool
git checkout dev

# 安装依赖
uv sync
```
